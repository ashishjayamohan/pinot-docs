# MOD

Modulo of two values

## Signature

> MOD(col1, col2)

## Usage Examples

```sql
select MOD(12, 5) AS value
from ignoreMe
```

| value |
| ----- |
| 2     |

```sql
select MOD(12, 2) AS value
from ignoreMe
```

| value |
| ----- |
| 0     |
