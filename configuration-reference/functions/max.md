# max

Get the maximum value in a group

## Signature

> MAX(colName)

## Usage Examples

These examples are based on the [Batch Quick Start](../../basics/getting-started/quick-start.md#batch).

```sql
select max(homeRuns) AS value
from baseballStats 
```

| value |
| ----- |
| 73    |
