# ABS

Absolute of a value

## Signature

> ABS(col1)

## Usage Examples

```sql
select ABS(-12.1) AS value
from ignoreMe
```

| value |
| ----- |
| 12.1  |

```sql
select ABS(12.1) AS value
from ignoreMe
```

| value |
| ----- |
| 12.1  |
